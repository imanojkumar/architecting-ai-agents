## 📖 About This Book

> *A comprehensive, practitioner-oriented guide to designing, building, and deploying AI agents — from foundational concepts to production-grade architectures.*

This book is intended for data scientists, ML engineers, software architects, and technically curious professionals who want to move beyond isolated model inference and into the world of **autonomous, reasoning AI systems**. It bridges theory and practice, covering agent design patterns, tool use, memory, multi-agent orchestration, and real-world deployment considerations.

*Content will be progressively added as the book evolves.*

## 🚀 Getting Started

### Prerequisites

- [R](https://www.r-project.org/) (≥ 4.3)
- [Quarto](https://quarto.org/) (≥ 1.4)
- [RStudio](https://posit.co/products/open-source/rstudio/) (recommended) or VS Code with Quarto extension

### Render Locally

Clone the repository and render the book:

```bash
git clone https://github.com/imanojkumar/architecting-ai-agents.git
cd architecting-ai-agents

# Render to HTML
quarto render

# Preview with live reload
quarto preview
```

The rendered book will be output to the `_book/` directory by default.
